# resource allocation through history ✨

**a living poster — [live at resource.gitcoin.co](https://resource.gitcoin.co)**

> 🖼️ **like it? [buy the physical poster](https://store.gitcoin.co/products/resource-allocation-history-framed-photo-paper-poster)** — the original "resource allocation through history" print, framed, from the gitcoin store.

An interactive remake of the Gitcoin "resource allocation through history" poster — 12,000 years of deciding *who gets what, and who decides*. Seven sibling posters are folded in as tabs: **[who decides](https://resource.gitcoin.co/power) · [how you eat](https://resource.gitcoin.co/live) · [what is true](https://resource.gitcoin.co/truth) · [who you are](https://resource.gitcoin.co/identity) · [who you trust](https://resource.gitcoin.co/trust) · [who keeps the record](https://resource.gitcoin.co/record) · [how you fight](https://resource.gitcoin.co/conflict)** — potlatch to quadratic funding, chiefdoms to forkable states, songlines to zero-knowledge proofs, wergild to peace bonds, each one explained.

Click around. It sparkles.

## features

- `index.html` — eight tabs, 551 mechanisms: **allocation** (81), **power structures** (80), **livelihood** (77), **truth** (68), **identity** (66), **trust** (60), **the record** (59), **conflict** (60)
- Every mechanism explained on click — what it is, where it came from, its tradeoffs
- Each era ends with the **unbuilt ones** ✦ — specs waiting for a builder
- Every tab has its own crawlable URL + social card, so links unfurl properly
- Light & dark themes, responsive layout, reduced-motion friendly
- Self-contained HTML files — no build step, no dependencies

## running locally

It's just files. Open `index.html` in a browser, or serve the folder with anything:

```sh
python3 -m http.server
```
